# API gRPC pour la Gestion des Comptes

Application Spring Boot exposant un service gRPC pour la gestion de comptes bancaires.

## Technologies

- Spring Boot
- gRPC
- Spring Data JPA
- H2 Database

## Démarrage

```bash
mvn spring-boot:run
```

## Opérations gRPC

- `allComptes` - Liste des comptes
- `compteById` - Compte par ID
- `saveCompte` - Création de compte
- `totalSolde` - Statistiques des soldes
